# Day02.基本DOS命令

## 基本命令之跳转
>
> 切换盘符： 盘符 + 冒号 ,例【 D: 】
>
> 查看目录下的目录和文件 ： dir
>
> 切换目录： cd + 位置 ,例【 cd /path 】
>
> /d参数可以使cd命令跨盘符跳转，例【 cd /d E:\IDE】可以实现跳转到E盘的目录下
>
> 相对位置跳转 : cd ..
>
## 基本命令之功能性命令
>
> 清理屏幕 ： cls
>
> 退出：exit
>
> 查看ip：ipconfig
>
> 查看ip和链接信息：ping + ip(或域名)
>
> 创建目录：md + 目录名
>
> 删除目录： rd + 目录名
>
>创建文件：cd > 文件名
>
>删除文件： del + 文件名
