+ 3.5 数组使用的简单案例


    + 数组求最值
      + 定义一个变量用于保存最终的最大值，从数组第一个数据开始
      + 从第二个数据开始辨遍历，如果该值大于变量，则替换变量的值

    + 数组反转
      + 方法一：逆向循环，插入新的数组
      + 方法二：依次前后交换数据
        ~~~java
        for(int i=0,j=arr.length-1;i<j;i++,j--){
            Object temp = arr[j];
            arr[j] = arr[i];
            arr[i] = arr[temp]
        }
        ~~~
